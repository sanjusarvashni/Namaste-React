Difference between a Library and Framework?

A library is a set of prewritten codes which can be imported into our project, and we use them to perform set of tasks or functions, and they are reusable.
whereas, a Frameworkis a set of prewritten codes that define the structure of a website or application.
in library, the developer calls the functions whenever required, and in framework, the developer has to mention the code wherever required by the framework , and its calls its functions.



What is CDN? Why do we use it?

CDN is a group of geographically distributed servers.To minimize the distance between the websites/users/other resources we want, we use cdn which will cache a copy of the content we want on each of their servers.
